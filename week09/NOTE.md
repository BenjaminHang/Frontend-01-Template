# HTML

##### 好用的 API
* compareDocumentPosition 是一个用于比较两个节点中关系的函数。
* contains 检查一个节点是否包含另一个节点的函数。
* isEqualNode 检查两个节点是否完全相同。
* isSameNode 检查两个节点是否是同一个节点，实际上在 JavaScript 中可以用 ===。
* cloneNode 复制一个节点，如果传入参数为 true，则会连同子元素做深拷贝。
